# 更新日志

## v0.2.0 - 2026-07-03

### 新增功能

#### ASMR 下载元数据与封面自动嵌入
- 下载 ASMR 作品时，自动将作品标题写入音频文件「专辑」标签
- 自动将声优名（逗号分隔）写入「艺术家」标签
- 自动将社团名写入「专辑艺术家」标签
- 支持三种封面尺寸选择：主封面（main）、缩略图（sam）、240x240 正方形
- 下载完成后自动将所选封面嵌入到所有音频文件（MP3 APIC / FLAC Picture / MP4 covr）
- 已完成的下载任务可随时更换封面，无需重新下载

#### 路径选择器
- 扫描管理新增目录浏览器对话框，支持图形化选择监控目录
- Windows 支持盘符列表浏览，支持逐级进入子目录

### Bug 修复

#### ASMR 下载监控目录下拉框为空
- **根因**：`/api/asmr/target-nodes` 端点通过 HTTP 自调用（`http://127.0.0.1:8000/local_node/api/watch-dirs`）获取监控目录列表，该请求因 token/端口/认证等问题静默失败，导致返回空列表
- **修复**：改为同进程内直接查询 local_node 数据库的 `watch_dirs` 表，消除 HTTP 自调用的所有失败点（token 生成、端口绑定、认证头、超时）
- 同时修复下载完成后触发扫描的同类问题：`_trigger_local_node_scan` 不再通过 HTTP POST 调用 `/local_node/api/scan`，改为直接调用 `run_scan()` 函数

#### 路径选择器滚动重绘
- 移除 DialogOverlay 的 `backdrop-blur-sm`（backdrop-filter 是滚动重绘的主要瓶颈）
- 为目录列表项添加 `v-memo` 指令跳过不必要的重渲染
- 为滚动容器添加 `contain: content` CSS 优化

### 技术改进

#### 新增文件
- `backend/app/plugins/asmr_one/tag_writer.py` — 基于 mutagen 的音频标签写入模块，支持 MP3/FLAC/MP4，独立于 local_node 的 metadata_editor 避免跨插件耦合
- `frontend/src/components/CoverPickerDialog.vue` — 封面选择对话框组件，支持三种封面候选预览与选择
- `更新日志.md` — 本文件

#### 后端变更
- `backend/app/plugins/asmr_one/models.py` — DownloadTask 新增 `metadata_json`（JSON）和 `cover_applied`（Boolean）字段
- `backend/app/plugins/asmr_one/database.py` — 新增 SQLite 列迁移逻辑（`ALTER TABLE ADD COLUMN`）
- `backend/app/plugins/asmr_one/downloader.py` — 下载完成后调用 `_apply_metadata_and_cover()` 写入标签与封面；`_trigger_local_node_scan()` 改为直接调用 `run_scan()`
- `backend/app/plugins/asmr_one/routers.py` — `DownloadCreate` 接收 `metadata` 字段；新增 `POST /downloads/{task_id}/apply-cover` 端点；`list_target_nodes` 改为直接查 DB

#### 前端变更
- `frontend/src/plugins/asmr_one/api.js` — 新增 `applyCover()` 函数
- `frontend/src/plugins/asmr_one/views/AsmrWorkView.vue` — 自动派生元数据、封面选择 UI、元数据预览面板
- `frontend/src/plugins/asmr_one/views/AsmrDownloadsView.vue` — 已完成任务新增「换封面」按钮
- `frontend/src/components/PathPickerDialog.vue` — 滚动性能优化
- `frontend/src/components/ui/dialog/DialogContent.vue` — 移除 backdrop-blur

### 已知限制
- 封面嵌入与标签写入尚未在真实 ASMR 下载中端到端验证（本机无 ffmpeg 生成测试音频）
- 如监控目录仍无法在下拉框显示，请检查扫描管理中是否已成功添加监控目录（前端节点 baseUrl 需包含 `/local_node` 后缀）
